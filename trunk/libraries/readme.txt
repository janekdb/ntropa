The following libraries are included in the Web Publishing System distribution because they are
required either for building the system or for running the system. Note that each
of these libraries is subject to the respective license; check the respective project
distribution/website before using any of them in your own applications.

* ant-ext.jar
- Copied from NetBeans 3.2.1 (originally ext/ant.jar)
- http://www.netbeans.org/

* ant-optional.jar
- Copied from NetBeans 3.2.1
- http://www.netbeans.org/

- Commons Collections 2.1
- http://jakarta.apache.org/commons/collections/

* commons-lang.jar

- http://jakarta.apache.org/commons/lang/

* commons-logging.jar, commons-logging-api.jar

- http://jakarta.apache.org/commons/logging/

* jakarta-oro.jar
- Jakarta ORO 2.0.5
- http://jakarta.apache.org/oro/index.html

* junit.jar

- http://junit.org

* mockobjects-core.jar, mockobjects-j2ee1.3.jar
